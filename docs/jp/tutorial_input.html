すべての入力オプション

ここでは，すべての入力オプションをまとめます． 入力のオプションとは，以下のPythonコードの pdc_sampler インスタンスの引数に対応します．

from aiphad import pdc_sampler

pdc  = pdc_sampler(page_type = "ternary_section",
                   input_data = "data.csv",
                   estimation = "LP",
                   gamma = 10,
                   sampling = "LC",
                   phase_id_option = {"FCC":0, "HCP":1, "BCC":2},
                   proposal = 3
                   )

入力には，以下のオプションがあります．

  • 入力データとしてcsvファイルを変数 input_data に入力する

  • 入力データを配列 X , y に代入する

それぞれの場合で，最低限入力が必要な変数は以下のとおりです． 〇は必須の項目，×は指定できない項目になります． △はどちらでも対応できる項目になります．

変数名

入力データがcsvのとき

入力データが配列のとき

説明

page_type

△

△

Webアプリ版に対応する相図の種類を入力します．ない場合，図の出力機能は利用できません．

input_data

〇

×

入力データをcsvファイルで入力するとき，ファイル名を入力します．

estimation

〇

〇

LPまたはLS法を指定します．

sampling

〇

〇

LC, MS, EA, RS法を指定します．

phase_id_option

×

△

ラベル名を指定します．

proposal

〇

〇

出力する提案点の数を指定します．

また，以下はオプションとして指定できる変数です．

変数名

入力データがcsvのとき

入力データが配列のとき

説明

gamma

△

△

相図推定手法において使われるrbfカーネルの \(\gamma\) を指定することができます．

parameter_constraint

△

△

１変数を固定する場合に利用します．１変数を固定する場合はTrueとします．デフォルトでは，Failseとなっています．

prev_point

△

△

１変数を固定する場合に，前回の提案点をリストとして入力します．

multi_method

△

△

OU, NEのどちらかを，proposalが２以上のときに，選択します．デフォルトではOUが選択されます．

NE_k

△

△

multi_methodでNEを選択した際に使用します．近傍の提案点は提案されなくなります．
