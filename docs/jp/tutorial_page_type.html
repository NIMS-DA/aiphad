Webアプリとの対応

page_type を利用することで，Webアプリ(https://aiphad.org/)と対応させることができます． このオプションを用いると，予測相図の描画，不確実スコアの描画が可能となります．

page_type は以下のように指定します．

pdc  = pdc_sampler(page_type = "ternary_section",
                   input_data = "data.csv",
                   estimation = "LP",
                   sampling = "LC",
                   proposal = 1
                   )

ここで指定できる page_type は以下の通りです．

  • “two_variables”: ２変数相図

  • “three_variables”: ３変数相図

  • “ternary_section”: ３成分系等温断面図

  • “ternary”: ３成分系相図

  • “quaternary_section”: ４成分系相図

input_data にcsvファイルを入力する場合， page_type に指定したものと対応したファイルを用いる必要があります．

３成分系等温断面図の例

ここでは基本的な例として，３成分系等温断面図(“ternary_section”)の計算を行う場合を利用して説明します． ３成分系等温断面図の計算を行うためには， page_type に”ternary_section”を指定します． この例では，入力はcsvファイルを与え，図を画面に出力します．

まず，以下のようにimportします． aiphad.aiphad_outputをインポートすることで，相図を可視化することができます．

from aiphad import pdc_sampler
from aiphad.aiphad_output import plot_out
import numpy as np

pythonコードを以下のように記述します．

pdc  = pdc_sampler(page_type = "ternary_section",
                   input_data = "data_ternary_section.csv",
                   estimation = "LP",
                   sampling = "LC",
                   proposal = 5
                   )

pdc.fit()
pdc.us()
plot_out(pdc)

proposals_index = pdc.proposals
proposals_X = pdc.proposals_X
proposals_us = pdc.proposals_us

print("proposal_index:", proposals_index)
print("proposal_X:", proposals_X)
print("proposal_us:", proposals_us)

belonging_index = pdc.belonging_index
belonging_X = pdc.belonging_X
belonging_probability = pdc.belonging_probability

for key, value in pdc.phase_id_dict.items():
        print("belonging_index_{}:".format(key),belonging_index[value])
        print("belonging_X_{}:".format(key),belonging_X[value])
        print("belonging_probability_{}:".format(key),belonging_probability[value])

ここで， input_data で指定したサンプルファイル”data_ternary_section.csv”は同じディレクトリにおきます．

このPythonコードを実行すると， plot_out(pdc) により初めに図が出力されます． 以下の３つの図が連続して出力されます．

  • Checked Points: すでに相が同定されている点が表示されます．

  • Uncertainty Score: 各点の不確実性スコアが表示されます．

  • Estimated: 推定された予測相図が表示されます．

Pythonコードで出力した proposal_index ， proposal_X は， 次の実験候補となる提案点となります． 同じ条件で，10点の候補をWebアプリ上で計算すると，緑の丸番号が提案点として出力されます． Python版とWebアプリ版で同じ場所が次の実験候補として提案されていることがわかります．

配列で入力する場合

page_type を指定した場合も配列としてデータを入力することができます． page_type に指定したものと対応した配列を入力してください．

pdc  = pdc_sampler(page_type = "ternary_section",
                   estimation = "LP",
                   sampling = "LC",
                   proposal = 1)

X = np.array([
    [0.00,1.00,0.00],
    [0.00,0.50,0.50],
    [0.00,0.00,1.00],
    [0.50,0.50,0.00],
    [0.50,0.00,0.50],
    [1.00,0.00,0.00],
    ])

y = np.array([-1,-1,0,1,2,2])

pdc.fit(X,y)
pdc.us()

plot_out(pdc)

  • X : 離散化された相図中の候補点を全て記載します．任意の次元を扱うことができます．

  • y : 離散化された相図中の候補点において，実験結果があり相が同定されている点には相のインデックスを0から順に整数で記入します．相が同定されていない点には-1を記入します．

出力結果は以下の通りです．

ラベルが入力されなかった場合は，この図のように0から連番で自動的にラベルが生成されます．

ラベルを入力するとき

配列で入力した場合，任意のラベルを入力することができます． 任意のラベルを入力する場合， phase_id_option を使用し，以下のように指定します．

pdc  = pdc_sampler(page_type = "ternary_section",
                   estimation = "LP",
                   sampling = "LC",
                   phase_id_option = {"a":0,"b":1,"c":2},
                   proposal = 1
                   )

X = np.array([
    [0.00,1.00,0.00],
    [0.00,0.50,0.50],
    [0.00,0.00,1.00],
    [0.50,0.50,0.00],
    [0.50,0.00,0.50],
    [1.00,0.00,0.00],
    ])

y = np.array([-1,-1,0,1,2,2])

pdc.fit(X,y)
pdc.us()

plot_out(pdc)

  • phase_id_option : 配列yの0,1,2の要素に対応するように，{“a”: 0 ,”b”: 1 ,”c”: 2 }とラベルを記述します．

出力のオプション

AIPHADでは３種類の図を出力することができます．

  • Checked_points : すでに相が同定されている点が表示されます．

  • Uncertainty_score : 不確実性スコアが表示されます．

  • Estimated : 推定された相図が表示されます．

図の出力は，以下の２種類の方法があります．

  • 図の出力をそのままデスクトップに表示する．

  • 図を画像として保存する．

図の出力は plot_out(pdc) で行います． オプションの指定がない場合，デフォルトでは図がそのままデスクトップに表示するようになっています．以下に例を示します．

pdc  = pdc_sampler(input_data = "example_ternary_iso.csv",
                   page_type = "ternary_section",
                   estimation = "LP",
                   sampling = "LC",
                   proposal = 3
                   )

pdc.fit()
pdc.us()

plot_out(pdc)

図を画像として保存する場合，以下のようにオプションを入力します．

pdc  = pdc_sampler(input_data = "example_ternary_iso.csv",
                   page_type = "ternary_section",
                   estimation = "LP",
                   sampling = "LC",
                   proposal = 3
                   )

pdc.fit()
pdc.us()

plot_out(pdc, elev_num = 30, azim_num = 45, save_fig = True, extension = "png", directory_name = "./out1")

各オプションの説明は以下の通りです．

  • save_fig : 図を画像として保存する場合True，図をデスクトップにそのまま出力する場合Falseを指定します．

  • elev_num : 3次元の図の場合，z軸から見た仰角を指定できます．

  • azim_num : 3次元の図の場合，x, y軸方向の方位角を指定できます．

  • extension : 画像の拡張子を指定できます．png, jpgまたはpdfを指定することができます．

  • directory_name : 保存先のディレクトリを指定します．このフォルダの中に page_type で指定した名前のフォルダが作成され，その中に図が出力されます．
