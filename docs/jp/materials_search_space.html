Materials Search Space Generator

材料探索空間をcsvで作成することで，AIPHADによる相図作成を実施することができます． 材料探索空間を作成することが手間になる場合に利用できる機能がAIPHADのWebアプリ版にあります．

Materials Search Space Generatorを利用することで，高次元であっても任意の材料探索空間を用意することができます． Webアプリ版では扱えない高次元空間もPython版では扱うことが可能です．

  • Multiple parameters: 複数の変数を離散化することで探索空間を用意することができます．２変数，３変数の場合は，Webアプリ版の2 variablesと3 variablesに対応します．

  • Multiple components: 組成を離散化することで多元系に対する探索空間を用意することができます．３元系，４元系の場合は，Webアプリ版のternary sectionとquaternary sectionに対応します．

  • Binaries: バイナリで与えられる探索空間を用意することができます．

候補数が多すぎる場合は出力されない場合があります． その場合はステップサイズを大きくするなど，探索候補を少なくする必要があります．
